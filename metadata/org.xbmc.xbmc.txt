Description:
It was designed for network playback, so you can stream your multimedia from 
anywhere in the house or directly from the internet using practically any 
protocol available.

Stable and nightly builds also available from the website.

Long build so developer APKs are provided for now.

possibly JDK 7 

build-essential default-jdk git curl autoconf unzip zip zlib1g-dev gawk gperf cmake ia32-libs

$$NDK$$/build/tools/make-standalone-toolchain.sh --ndk-dir=$$NDK$$ --install-dir=$STANDALONE_TOOLCHAIN_DIR --platform=android-14 --toolchain=arm-linux-androideabi-4.8 --system=$(uname -m)

cd $XBMC_DIR/tools/depends
./bootstrap
./configure --host=arm-linux-androideabi --with-toolchain=$STANDALONE_TOOLCHAIN_DIR --with-sdk-path=$$SDK$$ --with-ndk=$$NDK$$ --with-sdk=android-14 --prefix=$XBMC_DEPS_DIR
make -j $JOBS # they recommend giving it a high number here (e.g. 20) and it took quite a long time with my modest make -j3
cd $XBMC_DIR
make -C tools/depends/target/xbmc
make
make apk
