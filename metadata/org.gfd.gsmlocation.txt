Summary:Location backend
Description:
A  provider for
[https://github.com/microg/android_packages_apps_UnifiedNlp Nlp]
that uses a database of cell towers from [http://opencellid.org OpenCellID]
to make your current location available to other apps. 

Location calculation is done offline and doesn't report home.

Nlp needs to be installed already as System app.

*Database is licensed as CCBYSA.
