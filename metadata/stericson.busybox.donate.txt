Summary:BusyBox installer
Description:
'''Don't use the backup/restore feature. It'll knacker the system'''

Interface to install BusyBox 1.21.1 
(just ARM architecture for now as the ones can't be built) 
as well other versions off the internet (if you're brave).
Don't install the 1.21.0 version as it isn't in the apk.

BusyBox combines tiny versions of many common UNIX utilities into a single
small executable. It provides replacements for most of the utilities you
usually find in GNU fileutils, shellutils, etc. 

The app allows you backup and restore current binaries; to disable individual applets 
and see their help documentation.

The range of applets 
[https://raw2.github.com/linusyang/android-busybox-ndk/master/android_ndk_stericson-like available]
isn't identical to the upstream version.
.
