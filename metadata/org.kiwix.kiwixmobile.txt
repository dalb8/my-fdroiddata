Auto Name:Kiwix
Summary:Offline content reader
Description:
Kiwix is an offline content reader which can be used for Wikipedia as well as
other contents (Ubuntu documentation, WikiLeaks, WikiSource, etc).
Once you have downloaded the content file (potentially very large), you don't
need any connection to use it.

The Wikipedia files are accessible via the mobile [http://kiwix.org/m website];
the full list is [http://download.kiwix.org/zim here] either over http
or add '.torrent' to the file name to get a a bittorrent file.
Wiktionary English is about 800MiB.
.
